엑셀 입력 조회 시스템

이 시스템은 Firebase Firestore를 사용하여 데이터를 저장합니다.

Express 서버 없이도 정적 파일로 작동합니다.

입력 페이지 조회 페이지 로그인 옵션 설정

사용 방법

  1. 먼저 옵션 설정 페이지에서 계산 옵션을 설정하세요.
  2. 입력 페이지에서 데이터를 입력하고 저장하세요.
  3. 조회 페이지에서 필터링된 결과를 확인하세요.

주의사항

  • GitHub Pages는 정적 파일만 호스팅하므로 Express 서버 기능은 사용할 수 없습니다.
  • 모든 데이터는 Firebase Firestore에 저장됩니다.
  • 로컬 서버를 사용하려면 npm start 명령어를 실행하세요.
